# Installation

You can install the package using your favourite package manager, I prefer yarn. But you can choose whatever you like.

<Tabs>
<TabItem value='Yarn' label='yarn' default>

```bash
yarn add react-simple-date-range
```

</TabItem>
<TabItem value='npm' label='npm'>

```bash
npm install react-simple-date-range
```

</TabItem>
<TabItem value='pnpm' label='pnpm'>

```bash
pnpm install react-simple-date-range
```

</TabItem>
</Tabs>

## Requirements

- [Node.js](https://nodejs.org/en/) version 1.14 or higher.
- [React.js](https://reactjs.org) version 16.8.0 or higher.

## Typescript

Typescript is supported out of the box, please see our [types](types.md) section for more information on specific types.
