# Simple example

:::caution

This package is still in its **early stages**, bugs or weird behaviour is expected

:::

In order to get started, start by installing the package using a package manager of your choice. Please see our [installation](installation.mdx) steps for instructions.

This is the basis of how you would go about integrating a range component inside of your application.

## Creating your first range picker component

1. Start by creating a new `functional` React component

```jsx
const MyRangePicker = () => {
  return ();
};

export default MyRangePicker;
```

2. Now add the `DateRangePicker` component

```jsx {1,5-8}
import { DateRangePicker, DateRange } from 'react-simple-date-range';

const MyRangePicker = () => {
  return (
    <DateRangePicker
      value={range}
      onChange={(newRange) => setRange(() => newRange)}
    ></DateRangePicker>
  );
};

export default MyRangePicker;
```

3. Add react `useState` hook and provide the `state` and `setState` as props to the `DateRangePicker` component. If you're using TypeScript you can specify the `DateRange` type using `as`.

```tsx {1,5-8}
import { useState } from 'react';
import { DateRangePicker, DateRange } from 'react-simple-date-range';

const MyRangePicker = () => {
  const [range, setRange] = useState({
    start: null,
    end: null,
  } as DateRange);

  return (
    <DateRangePicker
      value={range}
      onChange={(newRange) => setRange(() => newRange)}
    ></DateRangePicker>
  );
};

export default MyRangePicker;
```

const [range, setRange] = useState({

} as DateRange);

## Datepicker only

Here is just a `DatePicker` component

```jsx {7}
import { useState } from 'react';
import { DateRangePicker, DateRange } from 'react-simple-date-range';

const MyRange = () => {
  const [date, setDate] = useState(new Date());

  return <DatePicker value={date} onChange={(newDate) => setDate(() => newDate)}></DatePicker>;
};

export default MyRange;
```
